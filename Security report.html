📡 گزارش امنیتی شبکه داخلی شرکت

اطلاعات کلی

📅 تاریخ گزارش: 2025-06-25

💻 تعداد دستگاه‌های شناسایی‌شده: 12

🌐 دستگاه‌های دارای HTTP/HTTPS فعال: 6

🔐 دستگاه دارای SSH فعال: 1 (Dropbear 2016.74)

🔍 بررسی آسیب‌پذیری‌ها

  • پورت‌های باز HTTP/HTTPS روی تجهیزات برند Yealink و Grandstream دیده شد.
  • دستگاه با آی‌پی 192.168.120.174 در برابر حمله Slowloris آسیب‌پذیر است.
  • نسخه‌ی سرور HTTP این دستگاه mini_httpd/1.19 بوده که مربوط به سال 2003 است و خطرناک تلقی می‌شود.

🛡 پیشنهادات امنیتی

  1. بروزرسانی دستگاه‌های دارای Dropbear SSH و mini_httpd.
  2. استفاده از فایروال جهت محدودسازی دسترسی به پورت‌های باز.
  3. فعال‌سازی لاگ و هشدار برای تلاش‌های ناموفق در ورود به SSH و HTTP.
  4. مرور مجدد تنظیمات دوربین‌های تحت شبکه و تغییر رمزهای عبور پیش‌فرض.

📎 یادداشت‌ها و پیوست‌ها

اسکن‌های انجام‌شده با ابزار Nmap شامل شناسایی پورت‌ها و نسخه‌ها بوده است:

PORT     STATE SERVICE VERSION
22/tcp   open  ssh     Dropbear sshd 2016.74 (protocol 2.0)
80/tcp   open  http    mini_httpd 1.19 19dec2003

🔸 آسیب‌پذیری Slowloris (DoS) محتمل (CVE-2007-6750)
🔸 CSRF، XSS ذخیره‌شده، DOM-based XSS یافت نشد
🔸 پاسخ کامل HTTP از دستگاه Grandstream دریافت شد:
    MAC Address: 00:0B:82:D1:72:63
    Device Type: احتمالا دوربین تحت شبکه (IP Camera)

⚠️ برخی دستگاه‌ها ترافیک HTTP بدون رمزگذاری SSL ارسال می‌کنند.
